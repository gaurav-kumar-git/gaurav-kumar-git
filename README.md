## Featured Projects

### [ToolLens](https://github.com/gaurav-kumar-git/ToolLens)
Attention-aware tool routing for long-context selection.

- Built a **single-forward-pass attention reranker**
- Selected retrieval-relevant attention heads without fine-tuning
- Introduced **online position correction** using anchor-token debiasing
- Achieved **5–6× lower reranking latency** than sliding-window RankGPT

### [Attention-Graph-Walk](https://github.com/gaurav-kumar-git/Attention-Graph-Walk)
Attention-guided retrieval for multi-hop reasoning in long contexts.

- Uses model attention patterns to discover relevant reasoning paths
- Focused on efficient retrieval for long-context problems

### [graph-enhanced-retrieval-qa](https://github.com/gaurav-kumar-git/graph-enhanced-retrieval-qa)
Graph-enhanced retrieval for multi-hop QA.

- Built on **2WikiMultihopQA**
- Trained with **Triplet Margin Loss** over **BGE-M3 embeddings**
- Achieved **0.95 MRR**, improving over a **0.73 DPR baseline**

### [bert-information-bottleneck](https://github.com/gaurav-kumar-git/bert-information-bottleneck)
Exploring compression trade-offs in BERT classifiers.

- Studied discriminative vs reconstructive learning
- Evaluated bottleneck sizes on **CLINC-150**
- Improved performance from **51% to 93.7%** under extreme compression
