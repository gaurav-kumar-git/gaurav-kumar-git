## Featured Projects

### [ToolLens: Attention-Aware Tool Routing](https://github.com/gaurav-kumar-git/ToolLens)

- Built a **single-forward-pass attention reranker** for long-context tool selection.
- Selected retrieval-relevant attention heads using only in-context examples.
- Developed **anchor-token-based online position correction** to reduce positional bias.
- Achieved competitive accuracy with **5–6× lower reranking latency** than sliding-window RankGPT.
- Evaluated against Direct Generation, ICR, QRHEAD, RankGPT, BM25, and ReasonIR across **7 benchmarks**.

### [Attention Graph Walk](https://github.com/gaurav-kumar-git/Attention-Graph-Walk)

- Explores attention-guided retrieval for multi-hop reasoning in long contexts.
- Uses model attention patterns to identify relevant nodes and reasoning paths efficiently.

### [Graph-Enhanced Retrieval for Multi-Hop QA](https://github.com/gaurav-kumar-git/graph-enhanced-retrieval-qa)

- Developed a graph-enhanced retriever on **2WikiMultihopQA**.
- Trained using Triplet Margin Loss over **BGE-M3 embeddings**.
- Achieved **0.95 MRR**, outperforming the DPR baseline of 0.73 MRR.

### [Quantifying Information Bottleneck in BERT](https://github.com/gaurav-kumar-git/bert-information-bottleneck)

- Studied discriminative and reconstructive learning under compressed BERT representations.
- Evaluated bottleneck dimensions of 8, 32, 64, and 128 on **CLINC-150**.
- Improved performance from **51% to 93.7%** under extreme 768d → 8d compression.

### Stance-Aware Transformers for Opinion Mining

- Fine-tuned `all-mpnet-base-v2` using contrastive learning.
- Reduced more than **2.5M candidate triplets** to a hard-sampled pool of 500K.
- Achieved stronger embedding separation than triplet and vanilla baselines.
